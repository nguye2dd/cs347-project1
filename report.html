CS347 Project 1

  1. My client is my mom, and she offers beauty services.
  2. All the images I used were ones that my mom took of her clients or that they sent to her. All of my content was made from scratch using HTML and CSS. I referenced many online resources to help me with styling and using HTML elements but all of the work was done by me.
  3. My site contains 5 pages: The homepage, which is in index.html The services/pricing page, which is in services.html The photo gallery, which is in gallery.html The about us page, which is in about.html The contact us page, which is in contact.html
  4. I have a header on each of my 5 pages that includes the business name and a navigation bar to the other pages.
  5. I have navigation links in my header, which redirects the user to other pages. I also have various navigation links in the home page, service page, and about us page that redirects users to the contact form to make an appointment. I also include navigation links that redirect to a google maps page for locating the business and some navigation links to phone numbers.
  6. I added media queries on pages that needed it to change the sizing and layouts as the screen gets smaller. Media queries were msotly used on my about page and on my navigation bar to make the buttons size proportionally.
  7. All of my pages share the same background color and the same layout, with the header with the navigation bar and the pages contents below it.
  8. I have plenty of images throughout my website. I believe my alt attributes for each image are descriptive enough, and in my photo gallery, when you click on a photo, a slideshow comes up with the full size image as well as using the alt tag as the images caption.
  9. All of my pages contains a favicon. It is a small icon that represents happiness from receiving beauty services, which is the goal of the business. I got this icon from an external website, which I give proper reference to at the bottom of each page.
  10. All my images, css and js files are all kept in their own respective folders in my project directory.
  11. I did not import or use any external/third party tools.
  12. I ran all of my HTML and CSS files to make sure that there are no errors or warnings
  13. My elements are pretty much aligned in my opinion. Nothing looks out of place.
  14. My pages uses multiple fonts including ones I imported from google fonts including Lobster, Chilanka, and Satisfy.
  15. The contact us page fulfills all of these requies. The form asks for users to enter their name, phone number, email address, preferred method of contact, desired service, a file uploader to upload any relevant images, and allows the client to specify any additional information. I also made the name, phone number, email, preferred method of contact required fields, denoted by a red asterick, and used JS to provide proper erro handling for them. If the uder submits the form, but these fields are missing, an alert message pop ups telling the user to fill in the missing required fields. If all requirements are satisfied and the client submits the form, an alert message pops up displaying a confirmation prompt on the webpage with the information the client entered.
  16. My about page contains an external link to a google maps link that shows the exact location of the business. At the bottom of the page, I also added a list of beauty product brands that my mom usually uses and recommends with links tot heir respective websites.
  17. I made sure all my files and source code was neatly organized, with proper indentation and used intuitive names for classes and id's
  18. On my gallery page, when hovering over a picture, a zoom animation happens. The picture zooms enlarges a bit and becomes a bit more transparent like a bubble. This was powered using css to scale the picture when hovering over it. I was also able to control the speed of the animation, but ended up setting it at 1 second. On my services page also, I created menus for specific services and pricing. I added an animaton on the borders using CSS.
